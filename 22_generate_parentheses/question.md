22\. Generate Parentheses
=========================

Given _n_ pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

For example, given _n_ = 3, a solution set is:

\[
  "((()))",
  "(()())",
  "(())()",
  "()(())",
  "()()()"
\]
Example 1:

Input: n = 3
Output: ["((()))","(()())","(())()","()(())","()()()"]
Example 2:

Input: n = 1
Output: ["()"]
 
### Difficulty:

Medium
